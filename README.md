# Быстрое создание папок и файлов для раскроя

Простое Windows-приложение для быстрого создания папок и заготовок файлов под программы раскроя (Vintech, Metalix, Unimach) с единообразным именованием по дате и времени.

## Для чего нужна эта программа

Программа помогает быстро и по единому шаблону создавать папку текущей даты и времени и файл-заготовку для загрузки в программу раскроя / ЧПУ-станок.

Примеры создаваемых имён:

- `2025-04-12/2025-04-12-1435/18 мм 2025-04-12-1435--ПЛАЗМА.rcam`
- `2026-02-06/2026-02-06-935/8 mm 2026-02-06-0935--HANSLASER.Dsp`
- `2025-11-03/2025-11-03-1542/2025-11-03-1542--ТРУБОРЕЗ.aucb`

Если за эту же минуту уже кто-то создал папку — программа ждёт начала следующей минуты и только тогда разрешает создание.

## Основные возможности

- Автоматическое формирование имени папки по текущей дате и времени (без секунд)
- Поддержка разных типов раскроя и расширений файлов:
  - Vintech / Лазер, Плазма, Газокислороная → `.rcam`
  - Metalix → `.Dsp`
  - Unimach / другая система → `.aucb` (без толщины)
- Округлённые углы окна и кнопки
- Плавное появление окна при запуске
- Кнопка закрытия в стиле современных приложений
- Защита от создания двух одинаковых папок в одну минуту

## Требования

- Windows (тестировалось на Windows 10 / 11)
- .NET Framework 4.7.2 или новее  
  (обычно уже установлен на большинстве современных Windows)

## Как пользоваться

1. Запустите программу
2. Выберите тип раскроя в выпадающем списке
3. Если нужно — введите толщину материала (например `16`, `8`, `25`)
4. Нажмите кнопку **OK**

Программа:
- создаст папку вида `ГГГГ-ММ-ДД`
- внутри неё создаст папку вида `ГГГГ-ММ-ДД-ЧЧММ`
- внутри неё создаст файл с понятным именем
- закроет себя

Если за эту минуту папка уже существует → кнопка будет ждать начала следующей минуты и покажет обратный отсчёт.
